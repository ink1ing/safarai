# v1.0.0

## Summary

- First distributable Safari AI Sidebar beta release with Release packaging, in-app update detection, and GUI-assisted installer flow.

## What Changed

- Added GitHub Copilot provider sign-in and model refresh support.
- Added in-app update check and installer entry in Settings.
- Switched beta packaging to use Release configuration.
- Added Safari toolbar toggle behavior for the host panel.

## Installation / Upgrade Notes

- Replace the existing app in `/Applications`.
- Launch the updated app once from `/Applications`.
- Confirm the Safari extension remains enabled in Safari Settings > Extensions.

## Known Issues

- Public distribution still depends on valid Developer ID signing and Apple notarization credentials being available on the release machine.
- Some internal OAuth-related code paths still emit compiler warnings, though builds succeed.

## Rollback

1. Quit Safarai.
2. Remove `/Applications/safarai.app`.
3. Reinstall the previous known-good DMG.
4. Re-open the app and confirm the Safari extension is enabled.
