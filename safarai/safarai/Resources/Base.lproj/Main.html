Safari AI Sidebar

Codex 账户与模型控制台

宿主 App 直接负责 Codex OAuth 登录、单账户凭证保存和默认模型选择。扩展侧所有页面动作都会使用这里保存的账户与模型。

当前状态

可以在 Safari 扩展设置中启用 `safarai` 扩展。

扩展已经启用，可以在 Safari 中打开扩展面板验证链路。

扩展尚未启用，请先在 Safari 设置中手动打开。

使用方式

  • 账号登录、退出、当前账号与模型选择均集中在 Safari 浏览器按钮面板里
  • 宿主 App 只负责承载 Safari Web Extension 与原生 OAuth / 请求能力
  • 扩展侧负责页面理解、问答、结构化提取与安全写入
  • 当前实现为单账户、单默认模型，全局共享
